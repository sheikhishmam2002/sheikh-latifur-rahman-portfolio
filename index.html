Comparative Evaluation of Machine Learning Algorithms for Multiclass Fault Diagnosis of Rotary Compressors Using Triaxial Vibration Signals
QPAIN, 2025
Published
Fault Diagnosis Vibration ML
IEEE Xplore
Compressors are essential in industrial operations, managing power generation, heating, ventilation, and airconditioning (HVAC) systems and refrigeration. The operation of compressor systems produces different kinds of hazards that create production stoppages along with financial setbacks and dangerous risks for personnel safety. Vibration-based monitoring is an effective method for early fault detection. The research analyzes multiple machine learning algorithms for compressor fault classification through vibration data analysis by implementing Random Forest, XGBoost, LightGBM, Support Vector Machine (SVM), and Multi-Layer Perceptron (MLP). The ADXL345 accelerometer recorded vibration data under four operational conditions: healthy, tilted, vibration-added, and with a loosely mounted sensor. Every 2 -second window produced 20 timedomain features (mean, standard deviation, skewness, kurtosis per axis, plus RMS and its mean, standard deviation, skewness, kurtosis) and subsequently z-score normalization. Among the analyzed models, XGBoost showed the highest classification accuracy at 89.10%, surpassing all other algorithms across key evaluation metrics. Random Forest followed with an accuracy of 86.32 %. Model performance was assessed using accuracy, precision, recall, and F1-score. Owing to its advanced diagnostic proficiency and relatively moderate computational requirements, XGBoost stands out as the most suitable candidate for on-device compressor fault monitoring.
A Comparative Study of Classical Machine Learning Models and CNN‑LSTM for Multiclass Time‑Series Classification in Hose Pipe Fault Diagnosis
ICCIT, 2025
Published
Time-Series CNN-LSTM Classification
Publication Link
This paper presents a comparative analysis of Clas sical and Deep Learning Models for vibration-based multi class fault detection in hose pipe systems. A dataset of 118,732 tri-axial accelerometer samples was collected across six fault states(Normal, Twisted, Blocked, Loosed, Leaked, Burst), and balanced to 114,354 samples (≈19,059 per class). Data were processed using sliding-window segmentation and standardized scaling. Classical classifiers (Random Forest, SVM, Logistic Regression, and kNN) were benchmarked against a hybrid CNN–LSTM model that combines 1D convolution with temporal modeling. Robustness was assessed by injecting Gaussian noise (σ = 0.4). Under clean conditions, all models exceeded 99% accuracy, with CNN–LSTM achieving 99.67%. With noise, Lo gistic Regression was most resilient (98.69%), Random Forest degraded sharply (90.05%), and CNN–LSTM retained 97.81%, offering the best accuracy–robustness trade-off. Reproducibility was ensured via fixed seeds, standardized preprocessing, and systematic results archiving. The comparative analysis highlights the most accurate and robust models among them for vibration based leakage detection in hose pipe systems.
AI‑Powered Secure e‑Voting System with Facial Recognition for Real‑Time Voting Authentication
QPAIN, 2026
Under Review
Security Face ID Computer Vision
Electronic voting systems face challenges with ensur ing security, accuracy, and transparency. Many current systems rely on manual identity verification or paper-based processes that are vulnerable to human error, delays, and trust issues. This paper presents a novel AI-driven e-voting system that integrates facial recognition with embedded hardware control and a web based dashboard for real-time monitoring. The system utilizes the InsightFace MobileFaceNet model for voter authentication and a Raspberry Pi for controlling the voting booth hardware. A Flask web application displays real-time voter participation and party vote counts, while the system ensures that voters can only vote once by using a secure local SQLite database. The system’s performance is evaluated through simulation results, demonstrating robust real-time operation and accurate vote tracking, with performance improvements over conventional vot ing systems. This approach highlights the potential of combining deep learning with embedded systems to provide a secure, scalable, and transparent e-voting solution.
